Loja Virtual

O HTML é uma linguagem de marcação utilizada para criar páginas acessadas a partir de um navegador. A característica principal dessas páginas é que elas utilizam o hipertexto para viabilizar a navegação. Todos os elementos que compõem uma página são posicionados por meio de comandos específicos da linguagem, denominados TAGs. Acompanhe um trecho da obra Desenvolvimento de software II. Inicie a leitura a partir de Criação e formatação de páginas Web com HTML/CSS.
